# RHYTHM-IA-Proyecto
Proyecto de videojuego de ritmo con inteligencia artificial

# ANTES DE PROBAR EL PROYECTO

Al ejecutar el proyecto por primera vez, te va a pedir acceso al internet con firewall, antes de darle permiso preciona para que tenga acceso a la red publica y privada, si no no va a poder detectar correctamente los movimientos de la camara y tendras que darle los permisos manualmente
